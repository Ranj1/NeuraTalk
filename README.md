# NeuraTalk

A voice-enabled chat application with AI-powered conversations.

## Project Structure

```
neuratalk/
├── neura-frontend/    # React frontend application
├── neura-backend/     # Node.js backend server
└── README.md          # Project documentation
```

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- MongoDB database

### Installation

1. **Backend Setup**
   ```bash
   cd neura-backend
   npm install
   ```

2. **Frontend Setup**
   ```bash
   cd neura-frontend
   npm install
   ```

### Running the Application

1. **Start Backend Server**
   ```bash
   cd neura-backend
   npm start
   ```
   Server runs on port 3400 (default)

2. **Start Frontend Development Server**
   ```bash
   cd neura-frontend
   npm start
   ```
   Frontend runs on port 3000 (default)

## Features

- Voice-enabled chat interface
- AI-powered conversations
- User authentication
- Chat history management
- Real-time messaging

## Technology Stack

- **Frontend**: React, CSS Modules
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **AI**: Google Gemini API
